🔧 Oogbesturing werkt niet? Volg deze stappen:

  1. Gebruik Chrome of Firefox (Edge/Safari werken vaak niet)
  2. Open via lokale webserver:
    • Installeer VS Code + Live Server extensie
    • Of voer in terminal uit: python -m http.server 8000 → Open http://localhost:8000
  3. Geef camera-toestemming (klik op "Toestaan" in de pop-up)
  4. Zorg voor goed licht (WebGazer heeft een duidelijke camera-input nodig)
