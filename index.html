⚠️ Oogbesturing werkt niet? Geen probleem!

Je kunt de app gewoon met je muis bedienen. Klik op de kaartjes om ze te selecteren.

Als je oogbesturing wél wilt proberen:

  1. Gebruik Chrome of Firefox (Edge/Safari werken niet)
  2. Open deze pagina via een lokale webserver:
    • Installeer VS Code + Live Server → Klik "Go Live"
    • OF: Voer in terminal uit: python -m http.server 8000 → Open http://localhost:8000
  3. Geef camera-toestemming als je browser daarom vraagt
  4. Zorg voor goed licht (WebGazer heeft een duidelijke camera-input nodig)
