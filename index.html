⚠️ Camera werkt niet? Geen probleem!

Je kunt de app gewoon met je muis of vinger bedienen.

Als je oogbesturing wél wilt proberen:

  1. Gebruik Chrome of Firefox op een LAPTOP/PC (mobiel/telefoon werkt NAUWELIJKS met WebGazer)
  2. Open via lokale webserver:
    • Installeer VS Code + Live Server → Klik "Go Live"
    • OF: Voer in terminal uit: python -m http.server 8000 → Open http://localhost:8000
  3. Geef camera-toestemming (klik op "Toestaan" in de pop-up)
  4. Zorg voor goed licht en kijk recht in de camera
